Inspetor de elementos

O Inspetor de Elementos no Google Chrome é uma ferramenta de desenvolvimento que permite aos desenvolvedores web examinarem e modificarem o código HTML, CSS e JavaScript de uma página da web em tempo real. Ele oferece uma visualização detalhada da estrutura e do estilo da página, permitindo que os desenvolvedores inspecionem elementos individuais, entendam como eles são renderizados e façam alterações para experimentar diferentes visuais e comportamentos.

Falando sobre tags

As tags em HTML são elementos fundamentais para a criação de uma página web. Elas são usadas para estruturar e formatar o conteúdo, bem como definir o tipo e a função de diferentes elementos na página. As tags são envolvidas por colchetes angulares (< e >) e são compostas por um nome que especifica o tipo de elemento.

  • Tags

    • Atributo das tags

    Os atributos das tags em HTML fornecem informações adicionais sobre os elementos e controlam seu comportamento. Eles são usados para modificar as características e o comportamento padrão de uma tag. Os atributos são especificados dentro das tags e geralmente têm um nome e um valor.

    Textos

    O principal propósito do texto em HTML é fornecer a estrutura básica e o conteúdo textual de uma página web. Ele permite criar parágrafos, títulos, listas, links, formatação de texto, imagens e muito mais. Com HTML, é possível estruturar e organizar o conteúdo de uma página, definir seções, aplicar estilos e formatos, e inserir elementos interativos.

    Listas ordenadas e não ordenadas

    -Listas Ordenadas

    Uma lista ordenada é usada para exibir itens em uma sequência específica ou em uma ordem numérica. Cada item é numerado de acordo com a posição na lista.

    -Listas Não Ordenadas

    Uma lista não ordenada é usada para exibir itens em uma sequência que não requer ordem específica. Cada item é marcado com um símbolo ou marcador, geralmente um ponto ou um pequeno quadrado.

    Links

    o elemento "link" é usado para vincular recursos externos, como folhas de estilo (CSS), ícones, arquivos de fonte e feeds de RSS, ao documento HTML. O uso correto do elemento "link" é essencial para garantir que o navegador exiba o conteúdo da página da maneira desejada.
